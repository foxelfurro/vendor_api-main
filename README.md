# 💎 VendorHub API (Backend)

Suministrador y gestor de inventario para una red de vendedores de joyería. Este servidor maneja la autenticación, el catálogo maestro y la lógica de transacciones seguras.

## 🚀 Tecnologías
* **Node.js** con **Express** (TypeScript)
* **PostgreSQL** (Hospedado en Neon)
* **JWT** para autenticación segura
* **Bcrypt** para encriptación de contraseñas

## 🛠️ Instalación y Setup
1. Instala las dependencias:
   ```bash
   npm install
2. Crear .env con el .env.example
3. npm run dev
